# Tools Directory

This directory is reserved for custom tools and scripts.

The bot's built-in tools are defined in `rc-bot/bot.py` - see OPERATIONS.md for how to add new tools.

You can use this directory for:
- Custom scripts that the bot might reference
- Helper utilities for maintenance
- Documentation about external tools
- Integration code for other systems

This directory is intentionally kept separate from the main bot code for organization.
